
# Python基础数字类
1. 数字类型
    - 常量数字
      - 直接表示出来的数字就是常量数字，即常量数
     - 数字类型
         - 十进制数  丨  123，154，4854
         - 浮点数   丨  1.21，0.214815，4e-12，4.32e22
         - 复数   丨  3+66j，4.5+7.89j
         - 二进制   丨  0b101101010011000011
         - 十六进制  丨  0x1fe，0xee2
         - 八进制   丨  0o567，0o277
    - 整数
    - 浮点数
    - 复数
    - bool作为数字使用


```python
a = 23451215
print(a)
```

    23451215
    


```python
a = 0o367
print(a)
```

    247
    


```python
a = 0xfed
print(a)
```

    4077
